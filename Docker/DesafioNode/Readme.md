# Desafio Docker - Nginx com Node

Desafio do curso Dev Full Cycle para criar um docker compose para proxy-reverso usando nginx e node.

Docker Compose é composto de 3 imagens
- Banco Mysql:5.7.16
- ngnix
- Node:19

## :memo: Instruções

- Terminal
```
docker-compose up -d --build
```

- Navegador
```
http://localhost:8080/
```
